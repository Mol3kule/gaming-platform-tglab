## Getting Started

First, run the development server:

```bash
npm run dev:server
# or
yarn dev:server
# or
pnpm dev:server
# or
bun dev:server
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

-   Authentication with JWT
-   Working REST API's
-   Clean design
-   Multi-language
-   Dark/Light mode
-   Websockets
-   Pagination, filtering
